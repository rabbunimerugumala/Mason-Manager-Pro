# **App Name**: Mason Manager Pro

## Core Features:

- Place Management: Create, read, update, and delete construction sites (places).
- Daily Worker/Labourer Count: Track and save daily counts of workers and laborers for each site, with plus/minus buttons.
- Payment Rate Setting: Set and update payment rates per worker/laborer per site.
- Payment Calculation: Display calculated payments for daily/weekly totals, updating instantly as counts/rates change.
- History Log: Maintain a history log per site with editable and deletable records, supporting user notes (CRUD).

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to provide a professional feel.
- Background color: Very light blue (#E8EAF6) to ensure readability and reduce eye strain.
- Accent color: Soft green (#8BC34A) for CTAs and highlights, providing a sense of confirmation and positivity.
- Body and headline font: 'PT Sans' sans-serif.
- Use large, readable buttons to enhance usability and accommodate touch inputs.
- Employ intuitive icons for navigation (calendar, gear/settings, edit, delete) to facilitate ease of use.
- Incorporate smooth visual transitions (feedback for save/edit/delete/navigation) to provide a polished user experience.
